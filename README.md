# EMC-EMS App

A basic Node.js + Express app connected to MongoDB Atlas, ready for deployment on Render.

## Setup
1. Clone the repo
2. Install dependencies:
   ```
   npm install
   ```
3. Create `.env` file:
   ```
   MONGO_URI=your-mongodb-atlas-uri
   PORT=3000
   ```
4. Run locally:
   ```
   npm start
   ```

## Deploy on Render
Push to GitHub and link your repo to Render.
